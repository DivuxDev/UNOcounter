# UnoCounter
webpage to take count of the points while playing UNO, it can be used mostly for any game 

###Database schema### 

jugadores

| id   | int(11)   |  auto_increment primary   |
| nombre    | varchar(255)   |   |


puntuaciones
| jugadorid   | int(11)   | 
| puntos    | int(11)   |   

| Left-Aligned  | Center Aligned  | Right Aligned |
|:------------- |:---------------:| -------------:|
| Row 1         | **Bold**        | Cell 3        |
| Row 2         | *Italic*        | Cell 6        |
| Row 3         | ~~Strike~~      | Cell 9        |
| Row 3         | [Link](dot.com) | Cell 9        |
